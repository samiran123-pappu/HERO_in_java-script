## Common Pattern
Most tracks use this structure:
- `md/Note.md` for concepts and VIP points
- topic example folder with focused `.js` files
- matching `.html` files to run examples in browser

## CONDITION Structure

- Legacy: `CONDITION/OLD/` folders

Legacy topics:
