# JavaScript For Loop

`for` uses three optional expressions:

```js
for (exp1; exp2; exp3) {
  // code
}
```

- `exp1`: runs once before loop
- `exp2`: loop condition
- `exp3`: runs after each iteration

## VIP Points
- `exp1`, `exp2`, and `exp3` are optional.
- If `exp2` is omitted, use `break` to avoid infinite loop.
