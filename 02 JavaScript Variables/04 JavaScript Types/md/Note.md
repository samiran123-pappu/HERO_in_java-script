# JavaScript Types

JavaScript has 8 main data types:
- String
- Number
- BigInt
- Boolean
- Object
- Undefined
- Null
- Symbol

## `typeof`
Use `typeof` to inspect value type.

## VIP Points
- Numbers in quotes become strings.
- `typeof null` returns `"object"` (historical JS behavior).
- `NaN` is type `"number"`.
