# JavaScript Date Reference (Quick)

## Core creators

- `new Date()`
- `Date.now()`
- `Date.parse()`
- `Date.UTC()`

## Display methods

- `toString()`, `toDateString()`, `toTimeString()`
- `toUTCString()`, `toISOString()`, `toLocaleString()`

## Deprecated methods (avoid)

- `getYear()`
- `setYear()`
- `toGMTString()`
