# JavaScript Where To

JavaScript can be placed:
- inside `<script>` tags in HTML
- in `<head>` or `<body>`
- in external `.js` files

## External Script Benefits
- better separation of concerns
- easier maintenance
- browser caching for faster load

## VIP Points
- `type="text/javascript"` is no longer needed.
- External `.js` files should not include `<script>` tags.
