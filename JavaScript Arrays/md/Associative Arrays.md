# php
```bash
// Define using the => operator
$user = [
    "name" => "Alex",
    "age" => 25
];

// Access by name
echo $user["name"];

// Loop through keys and values
foreach ($user as $key => $value) {
    echo "$key: $value";
}
```


# java
```bash
import java.util.HashMap;

HashMap<String, String> user = new HashMap<>();
user.put("name", "Alex");
user.put("age", "25");

System.out.println(user.get("name")); // Output: Alex
```
