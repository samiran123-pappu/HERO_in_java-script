# JavaScript Introduction

JavaScript can update HTML, change CSS, handle events, and process data.

## Core Capabilities
- Change HTML content (`innerHTML`, `innerText`)
- Change attributes (`src`, etc.)
- Change styles (`style.fontSize`, `style.display`)
- Show/hide elements

## VIP Points
- `document.getElementById()` is one of the most used DOM methods.
- Use `innerText` for plain text and `innerHTML` for markup.
