# JavaScript Output

JavaScript can display output in multiple ways:
- `innerHTML` / `innerText`
- `document.write()`
- `alert()` / `window.alert()`
- `console.log()`
- `window.print()`

## VIP Points
- Prefer DOM methods (`innerText`/`innerHTML`) over `document.write()` for real apps.
- `console.log()` is best for debugging.
- `window.print()` prints current page.
