# JavaScript String Methods

String methods return new strings; original strings are unchanged.

## Core Method Groups
- character access: `at`, `charAt`, `charCodeAt`
- extraction: `slice`, `substring`
- case conversion: `toUpperCase`, `toLowerCase`
- cleaning/padding: `trim`, `padStart`, `padEnd`
- replace/split: `replace`, `replaceAll`, `split`

## VIP Points
- `substr()` is deprecated.
- Methods do not mutate original string.
