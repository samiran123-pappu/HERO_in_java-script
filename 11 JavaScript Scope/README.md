# 11 JavaScript Scope

## Where Variables Live

Scope determines the accessibility (visibility) of variables. In JavaScript, there are three types of scope: Block scope, Function scope, and Global scope.

## ðŸ”‘ Key Takeaway

- **Global variables** are available everywhere.
- **Function variables** are only available inside the function.
- **Block variables** (`let`/`const`) are only available inside the block `{}`.
- Always use `const` or `let` to avoid polluting the global scope.
