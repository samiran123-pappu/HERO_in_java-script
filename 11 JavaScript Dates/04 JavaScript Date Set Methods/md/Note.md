# JavaScript Date Set Methods

Setters modify an existing Date object:

- `setFullYear()`, `setMonth()`, `setDate()`
- `setHours()`, `setMinutes()`, `setSeconds()`, `setMilliseconds()`
- `setTime()`

UTC setters are also available:

- `setUTCFullYear()`, `setUTCMonth()`, `setUTCDate()`, etc.

Date comparison works directly with operators because Date values convert to timestamps.
