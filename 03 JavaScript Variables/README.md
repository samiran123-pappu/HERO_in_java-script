# 03 JavaScript Variables

## Storing Data

Variables are containers for storing data values. In this track, we learn the three ways to declare them and when to use which.

## 📚 Topics

1.  **[01 Variables Basics](./01%20JavaScript%20Variables%20Basics/index.html)**
    *   What is a variable?
    *   Identifiers and assignment.

2.  **[02 Let](./02%20JavaScript%20Let/index.html)**
    *   The modern standard for reassignable values.
    *   Block scope explained.

3.  **[03 Const](./03%20JavaScript%20Const/index.html)**
    *   Values that cannot be reassigned.
    *   How `const` works with Arrays and Objects.

4.  **04 Var**
    *   The "old" way.
    *   Function scope vs Block scope.
    *   Why we avoid it today.

## 🔑 Key Takeaway

- Use `const` by default.
- Use `let` only if you know the value will change.
- Avoid `var`.
